# Functions

<a name="getSumArray"></a>

## getSumArray(array) ⇒ <code>Number</code>
Returns the sum of two-dimensional array.

**Kind**: global function
**Returns**: the sum of all numbers

| Param | Type |
| --- | --- |
| array | <code>Array</code> |

**Example**
```js
// returns 45
getSumArray( [[1,2,3],[4,5,6], [7,8,9]);
```
